class coinbase_agentkit.action_providers.x402.x402_action_provider.x402ActionProvider¶

Bases: ActionProvider[EvmWalletProvider]

Provides actions for interacting with x402.

This provider enables making HTTP requests to x402-protected endpoints with optional payment handling. It supports both a recommended two-step flow and a direct payment flow.
